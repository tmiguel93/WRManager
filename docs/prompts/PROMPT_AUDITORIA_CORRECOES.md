# PROMPT — AUDITORIA DE CORREÇÕES

Audite o projeto atual sem recriar tudo.

## Objetivo
Encontrar rapidamente problemas reais em:
- fluxo pós-corrida e avanço de semana
- pré-temporada que nunca termina
- patrocinadores duplicados
- proposta para pilotos/staff falhando
- branding pouco visível da equipe criada
- mistura de idiomas
- vazamento de elementos internos/debug

## Instruções
1. leia apenas os agentes relevantes
2. localize os pontos prováveis do bug
3. proponha correção mínima e segura
4. aplique a correção
5. adicione teste se o bug for crítico
6. reporte causa raiz e validação
