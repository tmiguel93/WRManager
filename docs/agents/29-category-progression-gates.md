# Agent Context — Category Progression Gates

## Objetivo
Controlar progressão entre categorias para impedir início em topo e dar sensação real de ascensão.

## Escopo
- tiers
- requisitos de desbloqueio
- convites
- prestígio
- reputação
- performance e saúde financeira

## Regras de execução
- bloquear top tiers no início
- usar progressão por mérito e oportunidade
- comunicar claramente o que falta para subir
- refletir convites de equipes maiores no mercado

## Entregáveis esperados
- regras de gate
- UI de bloqueio/desbloqueio
- card Road to the Top
- relação com global hub e mídia

## Critérios de aceite
- jogador começa em categorias menores
- subida parece merecida
- progressão fica clara e motivadora

## Observações
Ótimo lugar para conectar objectives de conselho, reputação e narrativa.
