# Agent Context — Team Identity and Theme

## Objetivo
Fazer a identidade visual da equipe criada impactar de forma nítida o produto inteiro.

## Escopo
- cores primary, secondary, accent e gradient
- contrastes automáticos
- preview ao vivo na criação da equipe
- aplicação nas telas principais e de corrida

## Regras de execução
- usar CSS variables ou design tokens dinâmicos
- validar contraste para texto, badges e botões
- nunca permitir combinação ilegível
- persistir tema por equipe e por save

## Entregáveis esperados
- theme engine por equipe
- preview de branding
- placeholders/livery base usando a paleta da equipe
- testes de contraste e persistência

## Critérios de aceite
- trocar cor altera dashboard, cards, standings, race HUD e negotiation screens
- identidade se mantém estável após reload
- UX fica mais personalizada e reconhecível

## Observações
Aplicar também em logos fallback e avatares de entidade para reforçar percepção visual.
