# Agent Context — Team Color Visual Propagation

## Objetivo
Garantir que a cor da equipe criada propague de verdade por todos os pontos visuais relevantes.

## Escopo
- dashboards
- cards
- standings
- results
- race viewer
- buttons
- placeholders
- team banners

## Regras de execução
- criar mapa explícito de superfícies que usam a paleta
- não limitar a mudança a um detalhe isolado
- validar contraste e legibilidade em cada contexto

## Entregáveis esperados
- checklist de propagação
- tokens por superfície
- testes visuais/regressão de tema

## Critérios de aceite
- o jogador percebe claramente sua equipe em qualquer tela
- cores não “somem” após navegação ou reload

## Observações
Este agente existe para atacar exatamente a reclamação de que a mudança de cor ainda não parece visual.
