# Agent Context — Real Logos and Asset Packs

## Objetivo
Permitir trabalhar com logos reais de forma controlada, via asset packs e override manual, sem travar o jogo.

## Escopo
- team logos
- supplier logos
- sponsor logos
- driver photos
- packs locais e registry

## Regras de execução
- asset pack deve ser opcional
- jogo precisa rodar perfeitamente sem mídia real
- cada asset deve guardar atribuição e status
- facilitar troca, revisão e fallback

## Entregáveis esperados
- formato de asset pack
- documentação de pastas e manifest
- carregamento com prioridade local
- tela de administração simples para debug interno

## Critérios de aceite
- usuário consegue enriquecer visualmente o jogo
- ausência de pack não quebra nada
- integração com logos próprios da equipe é compatível

## Observações
Bom caminho para ter visual forte sem acoplar scraping frágil ao core do projeto.
