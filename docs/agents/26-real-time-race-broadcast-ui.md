# Agent Context — Real-Time Race Broadcast UI

## Objetivo
Dar ao race viewer cara de transmissão manager premium, com foco em clareza e emoção.

## Escopo
- overlay principal
- leaderboard
- track map
- event ticker
- weather widgets
- mini-cards dos pilotos
- animações de posição e highlights

## Regras de execução
- usar motion sutil
- manter legibilidade em alta velocidade de simulação
- refletir identidade visual das equipes
- não sobrecarregar a tela

## Entregáveis esperados
- layout broadcast
- componentes reutilizáveis
- estados para pause/fast-forward
- microinterações elegantes

## Critérios de aceite
- assistir corrida é gostoso visualmente
- o usuário entende gaps, pneus, clima e eventos rapidamente
- o viewer parece parte central do produto

## Observações
Complementa o agente do viewer com foco exclusivamente em UX/UI.
